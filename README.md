# Calculadora de Coeficiente de Rendimento (CR)

Este é um mini-site desenvolvido para automatizar o cálculo do CR baseado na fórmula:
**CR = Σ(Mᵢ × Cᵢ) / ΣCᵢ**

## Estrutura do Projeto
- `app.py`: Servidor back-end em Python (Flask).
- `templates/index.html`: Interface do usuário.
- `static/style.css`: Estilização da página.
- `static/script.js`: Lógica do front-end e comunicação com o servidor.

## Como Executar
1. Certifique-se de ter o Python instalado.
2. Instale o Flask: `pip install Flask`.
3. Execute o servidor: `python app.py`.
4. Acesse no navegador: `http://localhost:5000`.

## Funcionalidades
- Adição dinâmica de disciplinas.
- Cálculo automático via API.
- Interface responsiva e moderna.
- Suporte a decimais com ponto ou vírgula.
