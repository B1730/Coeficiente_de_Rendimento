# Coeficiente_de_Rendimento.github.io
Projeto desenvolvido com apoio da ManusIA para auxiliar colaboradores no cálculo do Coeficiente de Rendimento. Atualmente, o processo é manual e sujeito a erros. A aplicação automatiza o cálculo, tornando-o mais rápido, prático e confiável, facilitando o dia a dia e reduzindo retrabalho.
